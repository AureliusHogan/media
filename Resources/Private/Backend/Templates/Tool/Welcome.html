• Analyse File index

    Detect whether a file is missing in the storage as example or whether they are duplicate records in the database.

    The same can be performed by a CLI command:

    cd {sitePath};
./typo3/cli_dispatch.phpsh extbase media:analyseIndex
    Tip! You can also configure a Scheduler Task which will send a report by email to if anything looks wrong.

  • Warm up cache

    Update some caching columns such as "number_of_references" to speed up the search.

    The same can be performed by a CLI command:

    cd {sitePath};
./typo3/cli_dispatch.phpsh extbase media:warmUpCache
